Prometheus: A Multi-Agent Framework for Intelligent Codebase-Level Operations

Zimin Chen 1*, Yue Pan2*, Siyu Lu3, Jiayi Xu2, Claire Le Goues4, Martin Monperrus3, He Ye2
1Sana AI, 2University College London, 3KTH Royal Institute of Technology, 4Carnegie Mellon University
*Equal Contribution

Abstract

Prometheus is a multi-agent framework built on FastAPI for intelligent operations over codebases. It supports tasks such as code review, bug resolution, and pull request evaluation. By modeling collaboration among agents as a state machine, Prometheus enables compositional reasoning and robust task workflows.

The framework integrates LLMs, retrieval-augmented generation, and testing backends to support both open-domain queries and structured agent pipelines. Prometheus is benchmarked on SWE-bench-lite, demonstrating competitive performance in program repair.
